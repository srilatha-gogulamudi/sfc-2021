1. if

2. else

if(condition) { if the contition returns true, then only the internal logic will be executed } else{ this block of code will be executed once the if condition fails (i.e., if condition is false) }

3. else if

if(condition) { if the contition returns true, then only the internal logic will be executed } else if(condition){ this block of code will be executed once the if condition fails (i.e., if condition is false) and it will check for the condition in else if block } else{ this will be executed if both the above conditions fail }
