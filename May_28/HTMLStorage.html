Using Web Storage, web applications can save the data locally (i.e., within User's Browser)
1. Local Storage: The data will be saved without any expiration time
localStorage.setItem(param1,param2);
param1: key (which will be accessed later)
param2: value (the data which the user wants to save)
By default, the value will be stored as a string.
To retrieve the saved value,localStorage.getItem(param1) --> return the param2 2. Session Storage: The data will be available only for 1 session (i.e, the data will be lost once you close the browser tab)
sessionStorage.setItem(param1,param2);
param1: key (which will be accessed later)
param2: value (the data which the user wants to save)
By default, the value will be stored as a string.
Note: To remove the value, localStorage.removeItem(param1);
